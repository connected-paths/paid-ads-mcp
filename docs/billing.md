# Billing

Paid Ads MCP is a hosted commercial product.

## Access model

- Access is tied to an active billing entitlement for the Paid Ads MCP product
- Billing is managed through the Connected Paths hosted account experience
- Billing status can affect MCP access for hosted users

## Product scope

Paid Ads MCP billing is separate from other Connected Paths products. A subscription for another product does not grant access to Paid Ads MCP unless explicitly stated by Connected Paths.
